#binary-tree

Simple project in C for binary tree creation.

## How to install

1. Install `gcc` (which is usually shipped with base development packages set):

- On **Debian**:
```bash
sudo apt install build-essential
```
- On **Arch Linux**:
```bash
sudo pacman -S base-devel
```

2. Make `build.sh` executable:

```bash
chmod +x build.sh
```

3. Run `build.sh`:

```bash
./build.sh
```
